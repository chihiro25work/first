鳴川ちひろのポートフォリオサイト①

created date 2025.04

制作概要・目的

転職活動で使用するポートフォリオ。
制作物や自分自身の強みなどを伝える。書類では伝わりにくい人柄についてもより深い理解を促し、一緒に働くイメージを持っていただく。

ターゲット

企業の採用担当者の方々

情報設計

トップページではインパクトを与えると同時にどんな人間かイメージがつきやすい構成にしたいと思い、文字を最小限にしました。 サブページでは、経歴や制作物だけではなく自身の思いやプライベートな部分を見せることにより、一人の人間として興味を持っていただけるような内容にまとめました。

デザイン

写真でインパクトを与えることと、温かみや活力、明るさなどのイメージを人々に抱かせる暖色を使用することで親しみやすさを演出ました。

制作期間

デザイン:2週間 HTML5,CSS3コーディング:5日間

使用ツール・言語

Figma / Photoshop / HTML5 / CSS3 / JavaScript
